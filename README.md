# Installation and Setup

1. Clone the repository with <br>
   `git clone https://github.com/anwrat/Kanban-Board.git`   
3. Navigate inside backend folder and Install dependencies <br> `npm install`
4. Create .env files inside the backend folder <br>
`PORT=5001`<br>
`MONGO_URI=your_mongodb_connection_string`<br>
`FRONTEND_URL=http://localhost:5173`
5. Start server with <br> `npm run dev`

6. Go inside frontend folder and install dependencies <br> `npm install`
7. Create a .env file and add <br> `VITE_BACKEND_API=http://localhost:5001`
8. Start the app with <br>`npm run dev`
